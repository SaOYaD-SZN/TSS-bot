## 🛠️ 𝖫𝗈𝖼𝖺𝗅 𝖣𝖾𝗉𝗅𝗈𝗒𝗆𝖾𝗇𝗍

Want to test or contribute to **ᏖᏕᏕ**? Follow these quick steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd YOUR_REPO_NAME
   ```

2. **Set up your environment and dependencies:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
   pip install -U discord.py
   ```

3. **Configure your Token:**
   * Open `bot.py`
   * Replace `YOUR_DISCORD_BOT_TOKEN` at the bottom with your private token from the [Discord Developer Portal](https://discord.com).

4. **Launch:**
   ```bash
   python bot.py
   ```
